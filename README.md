# 🎓 Student Performance Prediction System

This project is a machine learning-based system that predicts student academic performance using historical data.

## 📌 Problem Statement
It is difficult for teachers to identify weak students early. This system helps predict student performance based on various factors like study time, attendance, and previous grades.

## 🚀 Features
- Predict student final grades
- User-friendly web interface
- Uses machine learning models (Linear Regression, Random Forest)
- Real-time prediction using Flask

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Flask
- Matplotlib, Seaborn

## 📊 How It Works
1. Data is collected and cleaned
2. Features are selected and processed
3. Model is trained and evaluated
4. Web app is used for prediction

## ▶️ How to Run
1. Install required libraries: "pip install -r requirements.txt"
2.  Run the app:"python app.py"
3.  Open in browser:"http://127.0.X.001:6000"(for example)
